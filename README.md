## üß† What I Learned

Building and refactoring this project allowed me to master several advanced front-end development and architectural concepts:

- **Advanced Hook Logic:** I implemented complex state management using useState and handled side effects with useEffect. I focused heavily on manual interval management and cleanup to prevent memory leaks in components like the Pausable Timer.

- **Data Persistence Strategy:** Integration of LocalStorage was key to ensuring that user data remains consistent. I learned how to synchronize application state with browser storage to create a seamless user experience.
- **Semantic Structure and SOLID:** I moved away from generic containers to semantic HTML5 tags and followed clean code practices to ensure the repository is maintainable and scalable.
